Having used WordPress for many years, I did a full redesign of my webpage in 2026. I decided to build a new site from scratch as an experiment, and ended up building a simple, PHP-based CMS that renders markdown-files - inspired by [Grav](https://getgrav.org) and similar systems. The source code is available on [Github](https://github.com/bardlahn/bard.lahn.no) in case anyone is interested to take a look.
