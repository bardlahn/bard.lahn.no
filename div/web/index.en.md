This website is a place to collect my writings, a technological experiment and personal project, and a small contribution to an Internet run by people rather than tech giants. Inspired by [a](https://diagram.website) [lot](https://gossipsweb.net) [of](https://bengler.no) [great](https://www.are.na/shannon-mattern/poetic-web-i-inqsyxb6o) [people](https://jonathangray.org), I want to contribute to an open and [poetic](https://httpoetics.neocities.org) Internet, to promote [open digital tools](https://github.com/bardlahn/academic-open) and standards, and avoid giving [big tech more power](https://attac.no/digital-makt/) and profits than necessary.

Having used WordPress to run a website and blog for many years, I decided in 2026 to build a full website from scratch as an experiment. Inspired by [Grav](https://getgrav.org) and similar content management systems, I ended up building a site that stores content in static [Markdown](https://en.wikipedia.org/wiki/Markdown) files that are readable and editable without any special software required. I then built a PHP backend which serves content from these files in HTML/CSS/JavaScript.

Building my own site was also a way of making sure I do not rely on any third-party trackers or gather any form of personal data. The site does not use cookies. I use the open-source tool [GoatCounter](https://www.goatcounter.com) to gather basic statistics on site visits and use. The page does use Google Fonts for the time being, which means that IP address and some other basic information is shared with Google, but not used for tracking. Among other tools, I use the open-source PHP libraries [Parsedown](https://parsedown.org) and [Symfony Yaml](https://symfony.com/doc/current/components/yaml.html) to handle Markdown. The full source code is available on [Github](https://github.com/bardlahn/bard.lahn.no) for those who are interested to take a look.
